# ReadTogether
ReadTogether - A Spring Boot + Vue.js fullstack book review website project inspired by GoodReads

## Built with
- Docker - containerization
- Spring Boot - backend API
- Vue.js — website structure
- Vue Router — client-side routing
- Pinia — app state control
- Tailwind — styling
- PrimeVue - front end UI

## Starting the project
1. Clone the project
2. Add a ``.env`` in the project folder, follow ``.env.example`` to configure the project
3. Run ``docker compose build`` and wait for the project to build
4. Run ``docker compose up`` to start the project
5. Head to ``localhost:5173``, the project should be running now
